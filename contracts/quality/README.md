# Structural Quality Rules

This directory defines **tiered structural quality rules** used during inventory and certification.

- Tier 1: Structurally complete (AI-eligible baseline)
- Tier 2: Structurally inconsistent (limited eligibility with flags)
- Tier 3: Structurally unresolved (not eligible)

Rules are **contract-driven** and apply across federated systems.
They do not replace remediation work — they make it measurable and trackable.
